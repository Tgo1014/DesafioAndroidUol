# Desafio Android UOL

Aplicativo desenvolvido para processo seletivo no UOL, com objetivo de consultar a [Dribbble API](http://developer.dribbble.com/v1/) e criar listagem de shots.
# O que foi utilizado?
  - Kotlin
  - MVP
  - Retrofit
  - Picasso
  - GSON
  - Diferentes telas conforme orientação
  - Material Design
